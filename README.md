# Arduino Obstacle Avoidance Robot

## Overview
A simple autonomous robot capable of detecting and avoiding obstacles using an ultrasonic sensor.

## Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- DC Motors
- Chassis
- Battery

## Features
- Obstacle Detection
- Autonomous Movement
- Motor Control Logic
